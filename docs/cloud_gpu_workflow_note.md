## 클라우드 GPU 대여 (권장)

### RunPod 4090 실비
- Community Cloud: **~$0.34/hr**
- 이번 배치 (1,492 라인) 실행: **30-40분 × $0.34 = 약 $0.22**
- 연간 (주 2회 = 월 8회): **$24/년**
- 신품 4090 구매 ($1,800) 대비 **7,000-8,000시간 대여 가능**

### 로드 오버헤드 이슈

**Direct 업로드 (가정 → RunPod)**:
- 가정 업스트림 5-20 MB/s → 3.35 GB 업로드 5-10분
- 1회 overhead **8-15분** (업+다운)
- 짧은 배치일수록 비중 커짐

**Cloud-staged (핵심 최적화)**:

| 경로 | 속도 | 3.35 GB 전송 |
|------|-----|------------|
| 가정 → RunPod | 5-20 MB/s | 5-10분 |
| **AWS S3 → RunPod** (same region) | 200-500 MB/s | **30초** |
| **Cloudflare R2 → RunPod** | 150-400 MB/s | **40초** |
| GDrive → RunPod | 30-80 MB/s | 1-3분 |

→ **S3/R2 pre-stage**로 1회 overhead **8-15분 → 1-2분**, 약 **90% 단축**

### 하이브리드 운영 원칙

- **로컬 3060 Ti**: 단위 테스트 (L0301-0340 같은 ~20 WAV), 파라미터 튜닝
- **RunPod 4090**: 풀 배치 실행 (1,500+ WAV), Tier 2 포함 통합 실행
- **분기 기준**: 50라인 이하 = 로컬 / 300라인 이상 = RunPod

| 단계 | 소요 |
|------|-----|
| RunPod 계정 + 결제 | 5분 |
